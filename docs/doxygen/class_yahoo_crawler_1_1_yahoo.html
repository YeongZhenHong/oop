Detailed Description

Yahoo class inheriting Crawler class.

Allows an instance of the Yahoo class to be created, taking in user-input for search objective.

Contains get_article function, to retrieve required informational fields from an article.

Contains crawl, to conduct searching of keyword and output of a csv file of all the articles.
