Detailed Description

BotAPI class Defines a telegram bot object to allow TelegramBot class to interact with the database to insert and retrieve crawled data from the database.

Static variables cnx and cursor are used to open and close connections from each time the other classes attempt to make a database connection This is to ensure that the database connection is closed after each time the user is done with their commit
